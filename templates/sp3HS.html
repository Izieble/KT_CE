About this Project

This project aims to replace traditional paper timetables for courses in JKU with a webpage which offers the same functionality

Tha project aims to reduce the carbon footprint of the university with this approach

As a demo for the second Milestone of the project, this webpage serves as a proof of concept to test the QR-Codes for randomized data in SP3
